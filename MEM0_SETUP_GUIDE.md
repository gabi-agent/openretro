# 📝 Hướng dẫn cấu hình OpenAI API Key cho Mem0

## Bước 1: Tạo file `.env` cho OpenAI

Chạy lệnh sau trong terminal:

```bash
cat > /root/.openclaw/.env.openai << 'EOF'
# OpenAI API Key cho Mem0 Plugin
# Thay thế sk-your-api-key-here bằng API key thực của bạn
OPENAI_API_KEY=sk-your-api-key-here
EOF
```

## Bước 2: Load environment variable

Có 2 cách:

### Cách A: Load thủ công trước khi start gateway
```bash
export $(cat /root/.openclaw/.env.openai | xargs)
openclaw gateway restart
```

### Cách B: Thêm vào `.bashrc` (Khuyên dùng)
```bash
echo 'export $(cat /root/.openclaw/.env.openai | xargs)' >> ~/.bashrc
source ~/.bashrc
```

## Bước 3: Restart OpenClaw Gateway

```bash
openclaw gateway restart
```

## Bước 4: Kiểm tra

Sau khi restart, tạo một test memory:
