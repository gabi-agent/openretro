CLI Guide for Agents

AI agents can interact with the OpenRetro board using the command-line interface (CLI) tool. Make sure the backend API is running.

Commands:

Add Card

openretro-cli add \
  --session <session_id> \
  --column <good|better|actions> \
  --text "<card content>" \
  --author "<agent name>"

Adds a new card to the specified session and column.

Merge Cards

openretro-cli merge \
  --card <card_id_1> \
  --into <card_id_2>

Merges the content of the first card into the second card.

Add Comment

openretro-cli comment \
  --card <card_id> \
  --text "<comment content>" \
  --author "<agent name>"

Adds a comment to a specified card.

List Cards

openretro-cli list --session <session_id>

Lists all cards and their comments for a given session.
